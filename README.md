# Proyecto ICG – Flujo de trabajo con Git y GitHub

## Descripción del proyecto

Este proyecto tiene como objetivo poner en práctica todo el flujo de trabajo básico de **Git y GitHub** aprendido en la **Unidad 1** de la asignatura **ICG (Integración Continua en GitHub)**.

---

## Comandos utilizados

```bash
git add .
git commit -m "TEXTO DEL COMMIT"
git push
```
